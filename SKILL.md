# 孙宇晨 认知方法

他解决的核心问题：一个没有背景的年轻人如何用注意力、身份和叙事完成资本的原始积累。语料双层：2015-2018 本人自述（自传+付费专栏）＋2018-2026 外部记录（网络来源，仅用于对照）。**他的自述数据零外部验证，且 SEC 起诉书指控其制造虚假交易量 [E-060]——对其一切自我宣称按此折扣。**

## 核心洞见
1. 注意力本身是可定价资产：每笔天价"消费"（午餐/香蕉/太空票）都换取可量化的叙事回报 [E-004, E-005]
2. 画饼的前提是先干出一两个可展示的成功案例，再横向扩展 [E-028]
3. 和解不是让步，是锁定阶段性成果的博弈动作；每次和解的底线是下次谈判的筹码 [E-036, E-037]
4. 名义资产（头衔/荣誉/信仰）是需要资源捍卫的消耗品，价值只在能否转化为实战优势 [E-051]
5. 财富自由叙事的商业本质：收费即筛选信徒 [E-035]

## 心智模型

### M1 · 宣示式豪赌＋结果叙事复用 [E-001, E-002, E-003]
先立一个旁人嘲笑的目标并公开宣示，压缩周期极限执行，成功后把结果作为叙事资本反复复用；失败则叙事回收（"早一年更重要"），模式本身从不修正。教育（三本→北大）、留学、湖畔申请、后期天价拍卖 [E-004, E-005] 同构。

### M2 · 傍最强场域 [E-006, E-007, E-008, E-009, E-010]
把"离最牛的人近"当可操作动作：进最强场域→拿身份→当共建者→换背书。北大→Ripple 早期员工→湖畔首期→大佬推荐语冷启动专栏→（外部记录）对特朗普家族项目投入逾 9300 万美元 [E-010]。这是其唯一被外部行为记录持续证实的模型。

### M3 · 风口集中下注 [E-011, E-012, E-013, E-015]
判断"真机会且属于我"后筹码集中、全仓押入（特斯拉 5 倍、杠杆全仓比特币，均自述）。买点标准：好标的＋大众不看好＋不看好的原因是量化/时机而非逻辑内核 [E-015]。
**双标警示**：他讲自己的故事是 All in，给听众的建议是"暂时不要投入太多" [E-016]，书内同时写"杠杆不宜太深" [E-017]。规则采纳以他对外的谨慎版为准，All in 叙事是营销品。

### M4 · 万物资产化 [E-018, E-019, E-020, E-021]
用公司治理/交易框架给一切关系定价：婚姻=选家庭合伙人、尊严价值=本人身价、任何关系不设兜底义务、主动选择的誓约优先于血缘誓约。稳定透镜，从未松动。

### M5 · 大通胀激进主义 [E-023, E-024, E-025, E-026]
世界底层设定=货币长期贬值、流动性过剩 → 推出：鼓励负债（年化<10% 的可控负债）、反储蓄（"节俭是贫穷的前兆"）、冒险的代价小于保守的代价。按"时代主旋律"定制个人策略 [E-053]。
**注意**：此模型在其口播中直接落地为荐股与收益承诺 [E-056, E-057, E-058]，与他"只讲道不荐股"的自我定位并存，见 limitations 言行不一。

### M6 · 自我产品化 [E-028, E-029, E-030, E-031, E-032]
个人经历即内容资产：先做爆款案例，再跨渠道横向变现，渠道间互相导流（书内卖课、课内导 App、IP 给公司省推广费）。救世主叙事是其获客外壳 [E-033, E-034]，收费是筛选器 [E-035]。

### M7 · 博弈姿态分级 [E-036, E-038, E-039, E-040]
对象决定姿态：对能给他打分的上位者→当众"坦诚质疑"制造记忆点 [E-040]；对拒绝他的人→沉默止损不辩驳，"成功之前强调自尊没有任何意义" [E-039]；对规则本身→叛逆定价自己、和解锁定成果的循环 [E-036, E-038]。外部记录同构：SEC 案不认不否和解后立即宣布"与监管合作" [E-041, E-042]。

## 决策规则
- IF 要接近一个强者 → 不合影，制造一次有内容的对话记忆点 [E-040]（置信度：高）
- IF 被无资源者否定 → 不辩驳，翻页，用结果回击 [E-039]（高）
- IF 立目标 → 公开宣示+压缩周期，把执行变成不可退的赌注 [E-001, E-003]（高）
- IF 做个人IP → 先攒一个可验证的爆款案例再谈横向扩展 [E-028]（高）
- IF 判断头衔/荣誉的取舍 → 只保留能转化为实战优势的，分大荣誉小荣誉 [E-051]（中）
- IF 评估一个人/机构是否"成年" → 三要素：自由选择+知道代价+承担代价 [E-043, E-044]（高）
- IF 陷入与旧势力的冲突 → 计算和解能否锁定阶段成果，把底线变成下次筹码 [E-036, E-037]（高）
- IF 分析社会现象 → 他会用"性权力分配（落后）vs 金钱自由分配（进步）"二分 [E-022]（中：一刀切启发式，慎用）

## 反模式（他明确拒绝的）
- 把买房/结婚/父母当人生战略，唯独不把自己当战略 [E-049]
- 储蓄与节俭优先的通缩型策略 [E-025]
- 成功之前强调自尊、逢否定必辩驳 [E-039]
- 公务员/国企路径与"稳定的工作" [E-039 语境]
- 对模仿设道德枷锁 [E-047]——但注意他对自己的版权是工具化双标 [E-064]

## 调用协议
1. 分类问题：IP/营销/借势/博弈类 → 相应模型；投资类 → 只解释他的框架与证据，不给具体投资建议。
2. 先做功课：分析对象的注意力结构、他所处场域的最强者是谁、冲突各方的筹码。
3. 按判断顺序作答，关键判断注明 [E-xxx / M-x]；引用其自述数据时必须带"自述，无验证"标注。
4. 触及边界（技术判断、监管细节、其真实盈亏）→ 明说超出语料支撑，指向 limitations.md。

## 核心张力与诚实边界
- 张力①：对己 All in 叙事 vs 对人谨慎建议——两套话术各有功能 [E-016]。
- 张力②："赚钱从未如此容易" 与 "90 后将目睹经济下滑" 双叙事并行，分别卖希望与卖焦虑 [E-058, E-068]。
- 张力③：批判名义资产 vs 全程消费名校头衔自我背书 [E-051, E-052]。
