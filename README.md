# sunyuchen-skill · 孙宇晨认知方法 Skill

一个可调用的"认知方法 Skill"：提取孙宇晨（Justin Sun）公开材料中的心智模型、决策规则与反模式，供 AI Agent（Claude Code / Codex 等）在分析注意力营销、个人 IP、借势与博弈类问题时调用。**不是人格模拟，不做角色扮演。**

## 文件

| 文件 | 内容 |
|---|---|
| `SKILL.md` | 7 个心智模型、8 条决策规则、反模式、调用协议（每条结论带证据编号） |
| `evidence.md` | 78 条编号证据：逐字摘录 + 来源 + 证据等级 + 主张/行为/自述标记 |
| `limitations.md` | 孤证、矛盾记录、言行不一、材料缺口、诚实边界 |

## 方法论

由 person-to-skill 流水线炼化：带证提取 → 三重验证门（跨域复现/生成力/独特性，≥3 次出现+≥2 独立来源才算心智模型）→ 矛盾保留不调和 → 独立质检。孤证不入方法论，"他说的"与"他做的"分开记录，冲突时以行为为准。
