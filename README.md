# sunyuchen-skill · 孙宇晨认知方法 Skill

一个可调用的"认知方法 Skill"：从孙宇晨（Justin Sun）的公开材料中提取心智模型、决策规则与反模式，供 AI Agent（Claude Code / Codex 等）在分析注意力营销、个人 IP、借势与博弈类问题时调用。**不是人格模拟，不做角色扮演**；第三方独立研究作品，未经本人或关联机构授权、认可或背书。

## 7 个心智模型

| # | 模型 | 一句话 |
|---|---|---|
| M1 | 宣示式豪赌＋结果叙事复用 | 公开立一个旁人嘲笑的目标，压缩周期极限执行；成了反复复用作叙事资本，败了叙事回收，模式从不修正 |
| M2 | 傍最强场域 | 把"离最牛的人近"当可操作动作：进最强场域→拿身份→当共建者→换背书（其唯一被外部行为记录持续证实的模型） |
| M3 | 风口集中下注 | 判定"真机会且属于我"后筹码集中押入；买点=好标的＋大众不看好＋原因是量化/时机而非逻辑内核（注意：对己 All in、对人劝谨慎的双标已标注） |
| M4 | 万物资产化 | 用公司治理/交易框架给一切关系定价：婚姻=选家庭合伙人、尊严价值=本人身价、关系不设兜底义务 |
| M5 | 大通胀激进主义 | 世界底层设定=货币贬值+流动性过剩 → 鼓励可控负债、反储蓄、冒险的代价小于保守；按"时代主旋律"定制个人策略 |
| M6 | 自我产品化 | 个人经历即内容资产：先做爆款案例再横向变现，渠道互相导流；救世主叙事是获客外壳，收费是筛选器 |
| M7 | 博弈姿态分级 | 对象决定姿态：对上位者当众制造记忆点，对拒绝者沉默止损，对规则玩"叛逆定价自己、和解锁定成果"的循环 |

## 8 条决策规则

1. 要接近一个强者 → 不合影，制造一次有内容的对话记忆点（置信度：高）
2. 被无资源者否定 → 不辩驳，翻页，用结果回击（高）
3. 立目标 → 公开宣示+压缩周期，把执行变成不可退的赌注（高）
4. 做个人 IP → 先攒一个可验证的爆款案例，再谈横向扩展（高）
5. 判断头衔/荣誉取舍 → 只保留能转化为实战优势的，分大荣誉小荣誉（中）
6. 评估一个人/机构是否"成年" → 三要素：自由选择+知道代价+承担代价（高）
7. 陷入与旧势力的冲突 → 计算和解能否锁定阶段成果，把底线变成下次筹码（高）
8. 分析社会现象 → 他惯用"性权力分配（落后）vs 金钱自由分配（进步）"二分（中，一刀切启发式，慎用）

每条模型与规则在 `SKILL.md` 中均带 [E-xxx] 证据编号，可在 `evidence.md` 逐条回查逐字摘录与来源。

## 适用场景

**适合问它的问题：**
- 个人 IP / 内容账号冷启动怎么规划（它第一句会问你"有没有可验证的爆款案例"）
- 怎么接近行业大佬、进入更强的圈子、借势换背书
- 和平台/同行/机构发生冲突，怎么博弈、什么时候和解、和解换什么
- 要不要公开立一个激进目标，怎么把目标变成叙事资本
- 拆解某个公众人物/品牌的注意力营销打法
- 研究孙宇晨本人财富观、关系观的内在逻辑（含其言行不一的完整记录）

**不适合的场景：**
- 任何投资建议、代币推荐、收益预测——它只解释框架，不给操作
- 加密圈新闻闲聊、人物八卦（直接搜索更快）
- 技术判断、团队管理参考（语料无支撑或仅有负面二手证词，见 limitations.md）
- 冒充本人、生成背书、刷量或规避监管类请求——skill 内置拒绝

**使用前建议**先读一遍 `limitations.md`：此人言行分裂严重（教人 All in 自己劝人谨慎、教人模仿却告模仿者），skill 的规则一律**以他实际做的为准，而不是他嘴上说的**——这正是它比直接读他的书更有用的地方。

## 方法论

由 person-to-skill 流水线炼化：带证提取 → 三重验证门（跨域复现/生成力/独特性，≥3 次出现+≥2 独立来源才算心智模型）→ 矛盾保留不调和 → 独立质检。孤证不入方法论，"他说的"与"他做的"分开记录，冲突时以行为为准。
